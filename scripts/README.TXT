This folder contains scripts for building the Pilomar project.

buildpilomar is the original (now deprecated build script).
buildpilomar_buster32 is the script to build on BUSTER O/S (for RPi3B and RPi4B).
buildpilomar_bookworm64 is the script to build on BOOKWORM O/S (for RPI4B and RPI5B).
zip-pilomar is a backup script. It archives the main programs and data files for pilomar.
